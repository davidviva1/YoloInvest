## 🌍 Market Context（盘前宏观环境）

- SPY / QQQ 收盘价和涨跌幅
- VIX 走势
- 关键驱动（宏观事件/数据）
- **市场结构判断（量化依据）：**
  - Opening Range（前30min高低点）
  - 突破时间（开盘后多少分钟突破 OR）
  - VWAP 位置和斜率
  - ATR 消耗（今日振幅 / 20日 ATR）
  - 成交量分布（前30min占比 vs 正常值）
- 关键价位（阻力/支撑）
- 技术面结论（趋势日/区间日/混合）

## 🧠 行为模式诊断

识别当日出现的行为模式，每个模式包含：
- 触发条件
- 今日出现次数和具体位置
- 心理根源
- 为什么危险

常见模式：
1. 越跌越买（Averaging Down）— 致命级
2. 过度交易（Overtrading）— 严重级
3. 无计划交易（No Plan Trading）— 严重级
4. 报复交易（Revenge Trading）— 严重级
5. FOMO 追涨（Fear of Missing Out）— 中等级

## 🎯 明日交易预案

用 if-then 场景式，不是愿望清单：

### 场景 A：趋势日
确认条件 → 操作规则 → 止损 → 仓位 → 熔断条件

### 场景 B：区间日
确认条件 → 操作规则 → 止损 → 仓位 → 最大笔数

### 场景 C：数据日
确认条件 → 等待规则 → 仓位减半

### 触发器（贴在屏幕上）
- 🔴 止损触发 → 具体动作
- 🟡 连续盈利 → 强制休息
- 🟡 疲劳信号 → 自检问题
- 🔴 想补仓 → 念出口号

## 📝 一句话总结

狠一点。不是安慰自己，是给自己恐惧感。
格式：赚了/亏了多少 → 但真实风险是什么 → 如果重复会怎样
